# Wine Quality Exploratory Data Analysis (EDA)

This project performs an exploratory data analysis (EDA) on the White Wine Quality dataset from the UCI Machine Learning Repository. The objective is to understand the dataset, assess data quality, identify patterns, and explore the relationship between physicochemical properties and wine quality ratings.

## 📂 Project Structure

- `A1_2528711_Mor.ipynb` – Jupyter Notebook containing the analysis
- `A1_2528711_Mor-white.csv` – Dataset used for analysis
- `A1_2528711_Mor.pdf` – Final report
- `README.md` – Project documentation

## Analysis Performed

- Data loading and inspection
- Missing value and duplicate checks
- Descriptive statistical analysis
- Distribution analysis
- Correlation analysis
- Data visualization using histograms, boxplots, and heatmaps

## How to Run

1. Place the dataset in the project folder.
2. Open `A1_2528711_Mor.ipynb` in Jupyter Notebook.
3. Run all cells sequentially to reproduce the analysis and visualizations.

## Closing Notes

This analysis provides insights into the factors influencing wine quality and helps establish a foundation for future predictive modeling and machine learning applications.
